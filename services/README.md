# Services Assets

サービスごとのブランドアセットを配置するディレクトリです。

## 推奨構成

```text
services/
  service-name/
    logo/
      svg/
      svg-transparent/
      png/
      png-transparent/
      jpg/
    icon/
      svg/
      svg-transparent/
      png/
      png-transparent/
      jpg/
```

サービスごとに必要な形式だけを追加してください。

## サービス一覧

- `databuddy/`: Databuddy のブランドアセット
