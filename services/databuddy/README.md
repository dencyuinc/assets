# Databuddy Assets

Databuddy のブランドアセットを配置するディレクトリです。

## 推奨構成

```text
services/databuddy/
  logo/
    svg/
    svg-transparent/
    png/
    png-transparent/
    jpg/
  icon/
    svg/
    svg-transparent/
    png/
    png-transparent/
    jpg/
```

必要な形式のファイルだけを追加してください。
