## Asset Groups

`company/` は DENCYU Inc. 本体のブランドアセットです。ブランドカラー、会社ロゴ、会社アイコンなどの詳細は [`company/README.md`](company/README.md) に記載しています。

`services/` はサービス単位のブランドアセットです。共通の配置ルールは [`services/README.md`](services/README.md)、サービス固有の仕様は各サービス配下のREADMEに記載しています。

## Format Policy

基本構成は以下の形式別ディレクトリです。

- `svg/`: 標準SVG
- `svg-transparent/`: 透過SVG
- `png/`: 標準PNG
- `png-transparent/`: 透過PNG
- `jpg/`: JPG

必要な形式だけを追加できます。ベクターで扱える用途では SVG を優先してください。

## Usage Policy

### 許可される使用

1. 社内プロジェクト、資料、プレゼンテーションでの使用
2. DENCYU Inc. または各サービスの公式ウェブサイト、プロダクト、アプリケーションでの使用
3. 公式のマーケティング資料、パンフレット、プレス、営業資料での使用
4. 正式なパートナーシップ契約または事前承認に基づく第三者使用

### 禁止される使用

1. ロゴ、アイコン、文字、色、比率、余白、フォントなどの無断改変
2. DENCYU Inc. または各サービスの評判を損なう可能性がある文脈での使用
3. 政治的活動、違法行為、誤認を招く表示での使用
4. DENCYU Inc. または各サービスのアセットを第三者の商標、ロゴ、サービス名として登録または使用すること
5. 正式な契約や承認なしでの第三者による商用利用

### 利用時の注意

- アセットごとの色、サイズ、命名、余白ルールは配下READMEを確認してください。
- 透明背景が必要な場合は `*-transparent` ディレクトリのファイルを使用してください。
- ロゴやアイコンのアスペクト比を変更しないでください。
- 不明な用途や外部公開用途では、事前に確認してください。
