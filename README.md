このリポジトリには、DENCYU Inc.のブランドアセット（ロゴ、アイコンなど）が含まれています。

## ブランドカラー

DENCYU Inc.のブランドアイデンティティを維持するため、以下の公式カラーを使用してください。

- **Primary**: `#373a3d` - メインカラー、主要なデザイン要素に使用
- **Secondary**: `#c8c5c2` - セカンダリカラー、アクセントや補助的な要素に使用
- **White-like**: `#f3f2f3` - ライト背景や境界線に使用

## ファイル構成

### logo/

フルロゴ（ロゴマーク + テキスト）のファイルです。以下の形式で提供しています：

- **SVG**: ベクター形式（拡大縮小に最適）
  - `logo/svg/`: 標準版
  - `logo/svg-transparent/`: 透明背景版
- **PNG**: ラスター形式（高解像度）
  - `logo/png/`: 標準版
  - `logo/png-transparent/`: 透明背景版
- **JPG**: ラスター形式（圧縮版）
  - `logo/jpg/`: 標準版

各フォルダには以下のバリエーションが含まれています：
- `DENCYU-logo-dark-horizontal`: ダーク背景用・横型
- `DENCYU-logo-dark-vertical`: ダーク背景用・縦型
- `DENCYU-logo-light-horizontal`: ライト背景用・横型
- `DENCYU-logo-light-vertical`: ライト背景用・縦型
- `DENCYU-logo-white-horizontal`: 白背景用・横型
- `DENCYU-logo-white-vertical`: 白背景用・縦型

### icon/

アイコン（ロゴマークのみ）のファイルです。以下の形式で提供しています：

- **SVG**: ベクター形式（拡大縮小に最適）
  - `icon/svg/`: 標準版
  - `icon/svg-transparent/`: 透明背景版
- **PNG**: ラスター形式（高解像度）
  - `icon/png/`: 標準版
  - `icon/png-transparent/`: 透明背景版
- **JPG**: ラスター形式（圧縮版）
  - `icon/jpg/`: 標準版

各フォルダには以下のバリエーションが含まれています：
- `DENCYU-icon-dark`: ダーク背景用（ライトカラー）
- `DENCYU-icon-light`: ライト背景用（ダークカラー）
- `DENCYU-icon-white`: 白背景用（ダークカラー）

### 命名規則とカラーバリエーション

ファイル名の命名規則により、背景色とロゴ/アイコンの色の対応が決まります：

#### ロゴ（logo/）

- `*-dark-*`: **ダーク背景**用 → ロゴは**ライトカラー**（明るい色）で表示
- `*-light-*`: **ライト背景**用 → ロゴは**ダークカラー**（Primary: `#373a3d`など）で表示
- `*-white-*`: **白背景**用 → ロゴは**ダークカラー**（Primary: `#373a3d`など）で表示

#### アイコン（icon/）

- `*-dark`: **ダーク背景**用 → アイコンは**ライトカラー**（明るい色）で表示
- `*-light`: **ライト背景**用 → アイコンは**ダークカラー**（Primary: `#373a3d`など）で表示
- `*-white`: **白背景**用 → アイコンは**ダークカラー**（Primary: `#373a3d`など）で表示

#### レイアウト（ロゴのみ）

- `*-horizontal`: 横型レイアウト（幅広のスペースに適している）
- `*-vertical`: 縦型レイアウト（縦長のスペースに適している）

## 利用規約・ポリシー

### 許可される使用

1. **社内使用**: すべての社内プロジェクト、資料、プレゼンテーションでの使用
2. **公式ウェブサイト**: DENCYU Inc.の公式ウェブサイトやプロダクトでの使用
3. **マーケティング資料**: 公式のマーケティング資料、パンフレット、パブリシティでの使用
4. **パートナーシップ**: 正式なパートナーシップ契約に基づく第三者使用（事前承認が必要）

### 禁止される使用

1. **改変・加工**: ロゴの形状、色、比率、フォントなどを変更することは禁止されています
2. **不適切なコンテキスト**: 政治的な活動、違法な活動、またはDENCYU Inc.の評判を損なう可能性がある文脈での使用
3. **商標侵害**: DENCYU Inc.のロゴを他社の商標として登録または使用すること
4. **競合他社のプロモーション**: 競合他社のプロモーションや広告への使用
5. **非承認の第三者使用**: 正式な契約や承認なしでの第三者による商用利用

### 使用上のガイドライン

1. **最小サイズ**: ロゴは最小推奨サイズ（SVG版では64px、PNG版では128px）以上で表示してください
2. **クリアスペース**: ロゴの周囲には十分な余白を確保してください（ロゴの高さの20%以上）
3. **背景の選択**: 
   - **ダーク背景**には `*-dark-*` バージョンを使用（ロゴがライトカラーで表示されます）
   - **ライト背景**には `*-light-*` バージョンを使用（ロゴがダークカラーで表示されます）
   - **白背景**には `*-white-*` バージョンを使用（ロゴがダークカラーで表示されます）
   - 透明背景が必要な場合は `*-transparent` フォルダのファイルを使用
4. **アスペクト比の維持**: 
   - ロゴの縦横比を変更することは禁止されています
   - レイアウトに応じて適切なバージョンを選択してください：
     - **横型レイアウト**が必要な場合: `*-horizontal` バージョンを使用
     - **縦型レイアウト**が必要な場合: `*-vertical` バージョンを使用
   - 各バージョンでアスペクト比が異なる場合がありますので、指定されたバージョンを使用してください
5. **色の変更禁止**: 提供されたカラーバリエーション以外の色に変更することは禁止されています

### 品質管理

- **優先形式**: 可能な限りSVG形式を使用してください（スケーラビリティと品質のため）
- **解像度**: ウェブ用途ではSVG、印刷用途では高解像度PNG（300dpi以上推奨）を使用してください
- **フォールバック**: SVGがサポートされない環境では、適切なサイズのPNGを使用してください
